nova_utils.utils.string_utils.parse_time_string_to_ms(frame)[source]

Parse a time string or value to milliseconds.

This function takes a frame as input, which can be specified as a string in milliseconds (e.g., “100ms”), as a string in seconds (e.g., “2s”), as a float in seconds, as an integer in milliseconds, or as None. It converts the frame to an integer value in milliseconds.

Parameters:

frame (Union[str, int, float, None]) – The frame value to parse.

Returns:

The frame value in milliseconds. 0 if frame is None.

Return type:

int

Raises:

ValueError – If the input format for frame is invalid.

nova_utils.utils.string_utils.string_to_enum(enum, string)[source]

Convert a string to an enum value.

This function takes an enum and a string and returns the corresponding enum value. If the string does not match any enum value, a ValueError is raised.

Parameters:

  • enum (Enum) – The enum to search in.

  • string (str) – The string to convert to an enum value.

Returns:

The enum value corresponding to the input string.

Return type:

Enum

Raises:

ValueError – If the input string does not match any enum value in the specified enum.
